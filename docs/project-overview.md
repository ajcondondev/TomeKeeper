# TomeKeeper — Project Overview

## Purpose

TomeKeeper is a book tracking web application with automated end-to-end testing using Playwright.

## Goals

- Build a functional book tracker app
- Build a thorough Playwright test suite
- Maintain clean Git history and organized documentation throughout

## Tech Stack (planned)

| Layer | Technology |
|-------|-----------|
| Frontend | React (Vite) |
| Styling | Tailwind CSS or plain CSS |
| State / data | Local state + mock service layer |
| Testing | Playwright (E2E) |
| Version control | Git + GitHub |

## Core Features (planned)

- View a library of books
- Add a new book (title, author, status)
- Edit book details
- Delete a book
- Mark a book as Read / Currently Reading / Want to Read
- Filter/search books by status
